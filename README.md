# 血浆游离血红蛋白检测系统

基于机器学习的血浆游离血红蛋白(Free Hemoglobin, FHb)浓度预测系统，支持双溶液独立模型检测。

## 项目概述

本项目实现了一个高精度的血浆游离血红蛋白浓度预测系统，采用三波长光谱检测技术（375nm、405nm、450nm），结合支持向量机回归（SVR）机器学习算法，实现了对不同类型溶液的高精度浓度预测。

**当前版本**: 2.0.0
**更新日期**: 2026-04-05

---

## 核心功能

- **三波长光谱检测**：利用375nm、405nm、450nm三个波长进行吸光度测量
- **双溶液独立模型**：针对不同类型的溶液训练独立的预测模型
- **高精度预测**：支持溶液A和溶液B的独立预测
- **RESTful API**：提供完整的API接口，支持前后端分离
- **可视化界面**：提供友好的Web操作界面
- **数据模拟器**：支持双溶液模式的模拟测量

## 使用指南

### 检测面板

1. 选择检测模式：
   - **单溶液模式**：使用原始融合模型
   - **双溶液模式**：使用对应溶液的独立模型

2. 选择溶液类型：
   - **溶液A**：适用于高吸光度样品（375nm ≈ 1.97）
   - **溶液B**：适用于低吸光度样品（375nm ≈ 0.09）

3. 输入吸光度数据，点击"开始预测"

### 数据模拟器

支持双溶液模式的模拟测量：
- 单次模拟+预测
- 批量模拟+预测
- 生成双溶液测试数据集

### 示例数据

**溶液A（高吸光度）：**
| 波长 | 吸光度 |
|------|--------|
| 375nm | 1.9702 |
| 405nm | 0.6393 |
| 450nm | 1.0960 |

**溶液B（低吸光度）：**
| 波长 | 吸光度 |
|------|--------|
| 375nm | 0.0875 |
| 405nm | 0.0146 |
| 450nm | 0.0535 |

## ⚠️ 重要说明

### 数据限制

**当前训练数据量较少，可能存在以下问题：**

1. **模型泛化能力有限**
   - 当前每种溶液只有5-6个样本
   - 建议收集更多实验数据（至少20-30个样本/溶液）

2. **预测结果仅供参考**
   - 由于数据量限制，模型可能存在过拟合
   - 建议在实际使用前进行临床验证

3. **适用范围**
   - 溶液A: 浓度范围 0-0.3 g/L
   - 溶液B: 浓度范围 0-0.23 g/L
   - 超出范围预测可能不准确

### 建议改进

1. 增加实验样本量（每种溶液20-30个样本）
2. 每个浓度点进行3-5次重复测量
3. 收集更多种类的溶液数据
4. 添加异常值检测和过滤

## 训练自己的模型

### 1. 准备数据

将实验数据整理为Excel格式，包含以下列：
- 编号
- 浓度(g/L)
- 375nm
- 405nm
- 450nm

### 2. 修改配置

编辑 `双溶液数据预处理脚本.py` 中的溶液配置：
```python
self.solution_configs = {
    'solution_a': {
        'name': '溶液A',
        'file': '你的数据文件.xlsx',
        'concentration_col': '浓度g/L'
    }
}
```

### 3. 运行训练

```bash
python 双溶液模型训练脚本.py
```

训练完成后，模型将保存在 `models/` 目录。
