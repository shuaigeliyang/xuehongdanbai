# 血浆游离血红蛋白检测系统 - 机器学习算法层

## 项目概述

本项目实现了一个基于机器学习的血浆游离血红蛋白(FHb)浓度预测系统的算法层,采用三波长光谱检测技术(375nm、405nm、450nm),结合先进的机器学习算法,实现了高精度的浓度预测。

**完成时间**: 2026-03-16

## 📊 项目成果

### 模型性能

| 模型 | 测试集R² | 测试集MAE | 交叉验证R² |

- 三波长组合测量有效消除了样本背景干扰

### 3. 使用训练好的模型进行预测

```python
import pickle
import numpy as np

# 加载模型
with open('rf_model.pkl', 'rb') as f:
    model = pickle.load(f)
with open('scaler.pkl', 'rb') as f:
    scaler = pickle.load(f)

# 准备输入数据 (三个波长吸光度)
a375, a405, a450 = 0.5, 0.1, 0.8  # 示例值

# 创建特征
features = np.array([[
    a375, a405, a450,
    a405/(a375+0.001),  # A405/A375
    a450/(a405+0.001),  # A450/A405
    a450/(a375+0.001),  # A450/A375
    a405-a375,          # A405-A375
    a450-a405,          # A450-A405
    a375+a405+a450      # A_sum
]])

# 标准化并预测
features_scaled = scaler.transform(features)
concentration = model.predict(features_scaled)

print(f'预测浓度: {concentration[0]:.4f} g/L')
```

## 📈 技术细节

### 特征工程

从原始的3个波长吸光度特征扩展到9个特征:

1. **原始特征**: 375nm, 405nm, 450nm
2. **比值特征**: A405/A375, A450/A405, A450/A375
3. **差值特征**: A405-A375, A450-A405
4. **总和特征**: A_sum

### 模型优化

- **SVM回归**: 网格搜索优化超参数(C, gamma, epsilon, kernel)
- **随机森林**: 优化树数量(max_depth, n_estimators, min_samples_split等)
- **交叉验证**: 5折交叉验证确保模型泛化能力

### 数据集

- **总样本数**: 34
- **训练集**: 27样本 (80%)
- **测试集**: 7样本 (20%)
- **浓度范围**: 0 - 0.3 g/L
