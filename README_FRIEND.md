# 🩸 血浆游离血红蛋白检测系统

**基于机器学习的高精度血浆游离血红蛋白浓度检测系统**

---

## 🎯 这是什么？

这是一个**医疗检测辅助系统**，通过分析血浆样本的吸光度数据，自动计算游离血红蛋白浓度。

**核心技术：**
- 🤖 机器学习算法（Random Forest、SVR）
- 📊 三波长光谱分析（375nm、405nm、450nm）
- 🌐 Web 界面（React + FastAPI）
- 📈 数据可视化和统计分析

**适用场景：**
- 医疗实验室检测
- 科研数据分析
- 教学演示
- 算法验证

## 📊 系统特点

### ✨ 高精度检测
- 测试集 R² = 0.9980
- 平均绝对误差 MAE = 0.0033 g/L
- 均方误差 MSE = 0.000013

### 🎯 关键特征
- **A405/A375 比值**：最重要特征（82.63% 重要性）
- **三波长联合分析**：提高预测准确性
- **置信度评估**：自动给出预测可靠性

### 🔒 安全可靠
- 数据本地处理，不上传云端
- 支持离线使用
- 历史记录自动保存
