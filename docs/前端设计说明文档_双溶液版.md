## 🎯 系统概述

### 系统架构

本系统是基于机器学习的血浆游离血红蛋白浓度检测Web应用，支持**两种不同溶液**的独立预测：

```
┌─────────────────────────────────────────────────────────────┐
│                     前端界面 (React)                        │
│  ┌──────────────┐  ┌──────────────┐  ┌──────────────┐      │
│  │  溶液选择器  │→ │  吸光度输入  │→ │  预测结果显示 │      │
│  └──────────────┘  └──────────────┘  └──────────────┘      │
└───────────────────────────┬─────────────────────────────────┘
                            │ HTTP/REST API
                            ↓
┌─────────────────────────────────────────────────────────────┐
│              后端服务 (FastAPI + Python)                     │
│  ┌──────────────┐  ┌──────────────┐  ┌──────────────┐      │
│  │ 溶液A模型 SVR│  │ 溶液B模型 SVR │  │ 推理引擎     │      │
│  └──────────────┘  └──────────────┘  └──────────────┘      │
└─────────────────────────────────────────────────────────────┘
```

### 双溶液系统特点

| 特性 | 说明 |
|------|------|
| **独立模型** | 每种溶液使用独立训练的ML模型 |
| **自动选择** | 根据溶液类型自动加载对应的最佳模型 |
| **高精度** | 溶液A: R²=0.9657, 溶液B: R²=0.9987 |
| **科学性** | 基于朗伯-比尔定律的三波长检测 |

import React, { useState } from 'react';
import { SolutionType, SOLUTIONS } from './types';

export const SolutionSelector: React.FC = () => {

    </div>
  );
};

const SolutionCard: React.FC<{
  solution: SolutionConfig;
  selected: boolean;
  onClick: () => void;

  <div

### 3.2 吸光度输入组件

```typescript
// AbsorbanceInput.tsx
export const AbsorbanceInput: React.FC = () => {
  const [absorbance, setAbsorbance] = useState({
    a375: '',
    a405: '',
    a450: ''
  });

  const [errors, setErrors] = useState<{[key: string]: string}>({});

  const validateInput = (name: string, value: string): string => {
    const num = parseFloat(value);

    if (isNaN(num)) return '请输入有效数字';
    if (num < 0) return '吸光度不能为负数';
    if (num > 10) return '吸光度值异常';

    return '';
  };

  const handleChange = (name: string, value: string) => {
    setAbsorbance(prev => ({ ...prev, [name]: value }));
    setErrors(prev => ({ ...prev, [name]: validateInput(name, value) }));
  };

  return (
    <div className="absorbance-input">
      <h3>输入吸光度值</h3>

      <div className="input-grid">
        <InputField
          label="375nm"
          name="a375"
          value={absorbance.a375}
          onChange={handleChange}
          error={errors.a375}
          placeholder="例如: 1.9702"
        />
        <InputField
          label="405nm"
          name="a405"
          value={absorbance.a405}
          onChange={handleChange}
          error={errors.a405}
          placeholder="例如: 0.6393"
        />
        <InputField
          label="450nm"
          name="a450"
          value={absorbance.a450}
          onChange={handleChange}
          error={errors.a450}
          placeholder="例如: 1.0960"
        />
      </div>
    </div>
  );
};

const InputField: React.FC<{
  label: string;
  name: string;
  value: string;
  onChange: (name: string, value: string) => void;
  error?: string;
  placeholder?: string;
}> = ({ label, name, value, onChange, error, placeholder }) => (
  <div className="input-field">
    <label htmlFor={name}>{label}</label>
    <input
      type="number"
      id={name}
      name={name}
      value={value}
      onChange={(e) => onChange(name, e.target.value)}
      placeholder={placeholder}
      className={error ? 'error' : ''}
      step="0.0001"
    />
    {error && <span className="error-message">{error}</span>}
  </div>
);
```

### 5.2 API集成

```typescript
// api.ts
import axios from 'axios';

const API_BASE_URL = 'http://localhost:8000/api';

export interface PredictionRequest {
  a375: number;
  a405: number;
  a450: number;
  solution_type: 'solution_a' | 'solution_b';
  model_type?: 'rf' | 'svr' | 'best';
}

export const predictConcentration = async (
  request: PredictionRequest
): Promise<PredictionResult> => {
  try {
    const response = await axios.post<PredictionResult>(
      `${API_BASE_URL}/predict`,
      request
    );
    return response.data;
  } catch (error) {
    if (axios.isAxiosError(error)) {
      throw new Error(error.response?.data?.detail || '预测失败');
    }
    throw error;
  }
};

export const getSolutionsInfo = async (): Promise<Record<string, SolutionInfo>> => {
  const response = await axios.get<{solutions: Record<string, SolutionInfo>}>(
    `${API_BASE_URL}/solutions`
  );
  return response.data.solutions;
};
```

### 5.3 状态管理

```typescript
// usePrediction.ts
import { useState, useCallback } from 'react';
import { predictConcentration } from './api';

export const usePrediction = () => {
  const [loading, setLoading] = useState(false);
  const [result, setResult] = useState<PredictionResult | null>(null);
  const [error, setError] = useState<string | null>(null);

  const predict = useCallback(async (
    a375: number,
    a405: number,
    a450: number,
    solutionType: 'solution_a' | 'solution_b'
  ) => {
    setLoading(true);
    setError(null);

    try {
      const prediction = await predictConcentration({
        a375,
        a405,
        a450,
        solution_type: solutionType,
        model_type: 'best'
      });

      setResult(prediction);
      return prediction;
    } catch (err) {
      const message = err instanceof Error ? err.message : '预测失败';
      setError(message);
      throw err;
    } finally {
      setLoading(false);
    }
  }, []);

  const reset = useCallback(() => {
    setResult(null);
    setError(null);
  }, []);

  return { predict, loading, result, error, reset };
};
```
