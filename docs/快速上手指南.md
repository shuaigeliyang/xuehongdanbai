# 🎯 血浆游离血红蛋白检测系统 - 快速上手指南

**欢迎使用！** 本指南将帮助你在 5 分钟内启动系统并完成第一次检测。

---

## 📋 准备工作

### ✅ 第一步：检查软件环境

在使用系统前，请确保已安装以下软件：

1. **Python** (版本 3.8 或更高)
   - 检查方法：打开命令提示符，输入 `python --version`
   - 下载地址：https://www.python.org/downloads/
   - ⚠️ 安装时务必勾选 "Add Python to PATH"

2. **Node.js** (版本 18 或更高)
   - 检查方法：打开命令提示符，输入 `node --version`
   - 下载地址：https://nodejs.org/

### 💡 如果没有安装怎么办？
没关系！访问上面的下载地址，下载安装包后双击安装即可。安装过程大约需要 5-10 分钟。

---

## 🚀 启动系统

### 方法一：一键启动（推荐）✨

**最简单的方式！只需要双击！**

1. 找到项目文件夹中的 `启动系统.bat` 文件
2. 双击运行
3. 等待自动安装依赖（首次运行需要 3-5 分钟）
4. 看到以下提示表示启动成功：
   ```
   [OK] 系统启动完成！
   访问地址:
     - 前端界面: http://localhost:3000
   ```
5. 浏览器会自动打开系统界面

### 方法二：手动启动（进阶）

如果你想了解更多细节，可以手动启动：

**启动后端（在第一个命令提示符窗口）：**
```bash
cd backend
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python test_server.py
```

**启动前端（在第二个命令提示符窗口）：**
```bash
cd frontend
npm install
npm start
```

## 💡 如何使用系统

### 🎨 界面介绍

系统有 4 个主要功能标签页：

| 标签页 | 图标 | 功能说明 |
|--------|------|----------|
| **数据输入** | 📊 | 输入吸光度数据，进行浓度预测 |
| **数据模拟器** | 🧪 | 模拟光谱仪测量，生成测试数据 |
| **历史记录** | 📜 | 查看所有预测和模拟记录 |
| **统计分析** | 📈 | 查看模型性能对比和数据统计 |

---

### 📝 功能 1：浓度预测（最常用）

**适用场景：** 你有三波长吸光度数据，需要计算游离血红蛋白浓度

**操作步骤：**

1. **切换到"数据输入"标签页**

2. **输入吸光度数据**
   - **A375**: 375nm 波长的吸光度值
   - **A405**: 405nm 波长的吸光度值（通常是最重要）
   - **A450**: 450nm 波长的吸光度值

   示例数据：
   ```
   A375: 0.125
   A405: 0.175
   A450: 0.100
   ```

3. **选择模型**
   - 默认使用 Random Forest（推荐）
   - 也可以选择 SVR 模型

4. **点击"立即预测"按钮**

5. **查看结果**
   - 预测浓度：例如 `0.1523 g/L`
   - 置信度：`0.95`（95% 置信度）
   - 特征分析：显示各项比值和计算过程

**⚠️ 注意事项：**
- 吸光度值通常在 0.0 - 1.0 之间
- A405 通常 > A375（符合光谱特性）
- 如果置信度过低（< 0.80），建议重新测量

---

### 🧪 功能 2：数据模拟器

**适用场景：** 没有真实光谱仪数据，需要生成测试数据

#### 单次模拟

1. **切换到"数据模拟器"标签页**
2. **选择"单次模拟"**
3. **输入参数**
   - 浓度：例如 `0.15` (单位：g/L)
   - 噪声水平：`0.01`（默认值，通常不需要修改）
4. **点击"开始模拟"**
5. **查看结果**
   - 系统生成模拟的吸光度数据
   - 自动跳转到预测页面进行预测
   - 结果保存到历史记录

#### 批量模拟

1. **切换到"批量模拟"标签页**
2. **设置参数**
   - 样本数：例如 `10`（生成 10 组数据）
   - 浓度范围：`0.05 - 0.25`
3. **点击"生成批量数据"**
4. **查看和导出**
   - 表格显示所有生成的数据
   - 点击"导出 CSV"保存到本地

---

### 📜 功能 3：历史记录

**适用场景：** 查看之前的预测和模拟记录

**操作步骤：**

1. **切换到"历史记录"标签页**
2. **查看所有记录**
   - 显示时间、输入数据、预测结果
   - 点击"查看详情"查看完整信息
3. **清空记录**
   - 点击"清空历史"按钮
   - 确认后删除所有记录

---

### 📈 功能 4：统计分析

**适用场景：** 查看系统性能和数据分布

**查看内容：**

1. **模型性能对比**
   - Random Forest vs SVR
   - R²、MAE、MSE 指标
   - 推荐使用的模型

2. **特征重要性**
   - 哪些特征对预测最重要
   - A405/A375 比值是最重要的特征（约 82%）

3. **数据分布**
   - 浓度分布直方图
   - 统计摘要（均值、最大值、最小值等）

## 🔧 常见问题

### ❓ 问题 1：双击启动脚本后闪退

**原因：** 可能没有安装 Python 或 Node.js

**解决方法：**
1. 按 `Win + R` 打开运行对话框
2. 输入 `cmd` 打开命令提示符
3. 手动运行启动脚本：
   ```
   cd "项目路径"
   启动系统.bat
   ```
4. 查看错误信息并安装缺失的软件

---

### ❓ 问题 2：前端打不开（显示无法访问）

**原因：** 可能是后端服务未启动或端口被占用

**解决方法：**
1. 打开命令提示符
2. 检查后端是否运行：
   ```
   netstat -ano | findstr ":8000"
   ```
3. 如果没有输出，说明后端未启动，请重新启动系统
4. 如果有输出但不是我们的进程，运行 `停止系统.bat`

---

### ❓ 问题 3：点击"立即预测"没反应

**原因：** 浏览器缓存或网络问题

**解决方法：**
1. 按 `F12` 打开浏览器开发者工具
2. 切换到 "Console" 标签页
3. 查看是否有错误信息
4. 按 `Ctrl + F5` 强制刷新页面
5. 如果还有问题，重启浏览器

---

### ❓ 问题 4：显示"模型未加载"

**原因：** ML 模型文件不存在或不完整

**解决方法：**
1. 这是正常现象！系统会自动使用简化算法
2. 不影响基本功能使用
3. 如需完整功能，需要重新训练模型（参考高级文档）

---

### ❓ 问题 5：预测结果不准确

**可能原因：**
- 吸光度数据测量误差
- 样本处理不当
- 设备校准问题

**解决方法：**
1. 确认吸光度数据准确性
2. 重新测量样本
3. 检查光谱仪校准状态
4. 使用标准品进行验证
