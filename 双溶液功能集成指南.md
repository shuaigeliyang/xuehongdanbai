## 🎯 使用指南

### 1. 启动系统

#### 方式一：使用项目批处理文件（推荐）
```bash
# 双击启动文件
启动系统.bat
```

#### 方式二：手动启动
```bash
# 终端1：启动后端
cd backend
python main.py

# 终端2：启动前端
cd frontend
npm start
```

### 2. 访问系统
- **前端界面**: http://localhost:3000
- **后端API文档**: http://localhost:8000/docs

### 3. 使用双溶液检测功能

#### 步骤1：启用双溶液模式
在检测面板中，勾选"启用双溶液检测模式"复选框。

#### 步骤2：选择溶液类型
- **溶液A**: 2025年7月31日测量的标准溶液（高吸光度）
  - 适用于高吸光度样品
  - 模型精度: R²=0.9657
  - 示例数据: A375=1.9702, A405=0.6393, A450=1.0960

- **溶液B**: 2025年8月12日测量的实验溶液（低吸光度）
  - 适用于低吸光度样品
  - 模型精度: R²=0.9987
  - 示例数据: A375=0.0875, A405=0.0146, A450=0.0535

#### 步骤3：输入吸光度数据
- 375nm 吸光度
- 405nm 吸光度
- 450nm 吸光度

#### 步骤4：开始预测
点击"开始预测"按钮，系统将使用对应溶液的专用模型进行预测。

## 💡 使用建议

### 1. 溶液类型选择
- 根据您的实验样品类型选择对应的溶液
- 如果不确定，可以先尝试两种溶液，对比预测结果
- 选择R²值更高的溶液类型可以获得更准确的预测

### 2. 数据输入建议
- 确保吸光度数据准确（保留4位小数）
- 注意不同溶液的吸光度范围差异：
  - 溶液A：高吸光度（A375 ≈ 1.97）
  - 溶液B：低吸光度（A375 ≈ 0.09）

### 3. 预测结果解读
- 关注置信度（R²值）：越接近1表示预测越准确
- 查看溶液信息：确认使用了正确的溶液模型
- 对比历史数据：验证预测结果的合理性

## 🚨 注意事项

### 科学原理
1. **为什么需要双溶液？**
   - 两种溶液的吸光度水平差异巨大
   - 溶液A平均吸光度：1.9702（375nm）
   - 溶液B平均吸光度：0.0875（375nm）
   - 差异高达1.8827！

2. **分开训练的优势**
   - 每种溶液使用专门的模型
   - 避免不同溶液间的相互干扰
   - 提高预测精度和可靠性

### 技术限制
- 双溶液模式仅支持SVR模型（已优化）
- 单溶液模式支持RF和SVR模型
- 需要确保对应的溶液模型文件已加载
