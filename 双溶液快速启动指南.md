## 🎯 使用双溶液检测功能

### 1. 进入检测面板
- 启动后，默认显示"检测面板"标签页
- 确认右上角显示"✓ 正常运行"

### 2. 启用双溶液模式
- 在左侧表单顶部，找到"溶液模式"部分
- 勾选"启用双溶液检测模式"复选框
- 系统会显示溶液选择器

### 3. 选择溶液类型
- **溶液A**：适用于高吸光度样品（2025年7月31日测量的标准溶液）
  - 精度：R²=0.9657
  - 示例：点击"使用示例数据"会填充高吸光度测试数据

- **溶液B**：适用于低吸光度样品（2025年8月12日测量的实验溶液）
  - 精度：R²=0.9987
  - 示例：点击"使用示例数据"会填充低吸光度测试数据

### 4. 输入吸光度数据
- **375nm 吸光度**：输入对应波长的吸光度值（保留4位小数）
- **405nm 吸光度**：输入对应波长的吸光度值（特征峰）
- **450nm 吸光度**：输入对应波长的吸光度值

### 5. 开始预测
- 点击"开始预测"按钮
- 等待预测完成（通常在1秒内）
- 查看右侧的预测结果

### 6. 查看结果
预测结果会显示：
- **预测浓度**：血浆游离血红蛋白浓度（g/L）
- **置信度**：预测的可信程度（百分比）
- **溶液信息**：使用的溶液类型和描述
- **模型性能**：R²、MAE等性能指标

## 📊 溶液选择指南

### 溶液A（标准溶液）
**适用场景：**
- 高吸光度样品（A375 > 1.0）
- 2025年7月31日测量的标准溶液
- 需要较高精度（R²=0.9657）

**示例数据：**
```
样本编号: SAMPLE-A-001
A375: 1.9702
A405: 0.6393
A450: 1.0960
```

### 溶液B（实验溶液）
**适用场景：**
- 低吸光度样品（A375 < 0.5）
- 2025年8月12日测量的实验溶液
- 需要超高精度（R²=0.9987）

**示例数据：**
```
样本编号: SAMPLE-B-001
A375: 0.0875
A405: 0.0146
A450: 0.0535
```

## 🛠️ 故障排除

### 问题1：后端无法启动
**可能原因：**
- Python环境缺少依赖包
- 模型文件缺失

**解决方案：**
```bash
# 安装依赖
pip install -r backend/requirements.txt

# 检查模型文件
ls backend/../models
```

### 问题2：前端无法连接后端
**可能原因：**
- 后端未启动
- 端口被占用

**解决方案：**
```bash
# 检查后端是否运行
curl http://localhost:8000/health

# 如果端口被占用，修改端口
# 在 backend/main.py 中修改 port=8000 为其他端口
```

### 问题3：双溶液预测失败
**可能原因：**
- 模型文件未加载
- 溶液类型选择错误

**解决方案：**
```bash
# 运行验证脚本
python backend/verify_dual_solution.py

# 检查模型文件
ls backend/../models/solution_*
```
