## 📞 技术支持

### 🔍 问题诊断

**问题 1：启动失败**
- 检查 Python 版本：`python --version`
- 检查 Node.js 版本：`node --version`
- 查看错误日志

**问题 2：界面无法访问**
- 检查后端状态：http://localhost:8000/health
- 检查端口占用：`netstat -ano | findstr ":8000"`
- 重启系统

**问题 3：测试失败**
- 运行诊断：`python tests/auto_test.py`
- 查看测试报告
