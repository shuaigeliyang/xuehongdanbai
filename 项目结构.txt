     📌 快速开始指南
═══════════════════════════════════════════════════════════════

  第一步：查看文档
    └─ 打开 README_FRIEND.md 了解项目概况（3 分钟）

  第二步：检查环境
    ├─ Python 3.8+  → python --version
    └─ Node.js 18+  → node --version

  第三步：启动系统
    └─ 双击运行：启动系统.bat（首次需 3-5 分钟安装依赖）

  第四步：开始使用
    ├─ 前端界面：http://localhost:3000
    ├─ API 文档：http://localhost:8000/docs
    └─ 健康检查：http://localhost:8000/health

  第五步：运行测试
    └─ python tests/auto_test.py

═══════════════════════════════════════════════════════════════
     🎯 核心文件说明
═══════════════════════════════════════════════════════════════

  【启动相关】
    • 启动系统.bat         → 一键启动（自动安装依赖）
    • 停止系统.bat         → 一键停止（清理端口占用）

  【后端核心】
    • test_server.py       → 主服务器（适合测试和演示）
    • model_inference.py   → ML 模型推理（支持 RF + SVR）
    • simulator.py         → 光谱仪模拟器（朗伯-比尔定律）
    • *.pkl                → 训练好的模型文件（可直接使用）

  【前端核心】
    • App.js               → 主应用（标签页导航）
    • DetectionPanel.js    → 数据输入界面
    • SimulatorPanel.js    → 模拟器界面（单次 + 批量）
    • HistoryPanel.js      → 历史记录查看
    • StatisticsPanel.js   → 模型对比和统计图表

  【测试工具】
    • auto_test.py         → 自动化测试（18 个测试用例）
    • 测试用例.md          → 手动测试清单
